Tipografia

A tipografia transmite nossas definições de tamanhos e espaçamentos e deve ser utilizada respeitando a hierarquia dos elementos na interface.

Elementos

Textos

Para utilizar textos em negrito temos duas formas uma delas é utilizando as tags <b> e a outra é utilizando a tag <strong>. A diferença delas é que a tag b não tem uma importância semântica tão grande quanto a tag Strong, por outro lado a tag Strong quer dizer que esse texto realmente tem importância para o browser.

Agora para utilizar o itálico é com a tag <i> e para dar uma ênfase a uma frase se usa a tag <em>. Na pratica é a mesma coisa que o strong e o b, a tag <em≶ tem uma importância maior do que a tag <i>.

Falando agora de textos sublinhados, para realizar esse sublinhamento se utizou da tag <u> oque torna muito parecido com a tag <a> , porém como já sabemos a tag <a> não significa um texto sublinhado e sim um link, então a tag <u> para tal uso não é recomendado além de seu uso semântico estar errado, para sublinhar um texto é mais recomendado utilizar o CSS com o text-decoration utilizando uma Span ( <span> ) que não possui valor semântico algum. a tag <u> é mais utilizada para mostrar que algo esta escrito errado, desse jeito em conjunto de um código de CSS ( text-decoration:red wavy underline; ).

Sobrescrito e subscrito

3 2 é igual à 9. explicação: para utilizar o 2 foi utilizada a tag <sup>, que faz esse efeito muito usado para escrever elementos da tabela periódica ou cálculos matemáticos como no exemplo. Para fazer algo semelhante temos a tag <sub> que tem o mesmo valor semântico porem ao invés de colocar o número encima coloca em baixo, exemplo H2O.

Abreviacões

Para criar abreviações se utiliza a tag <abbr>, porque ela mostra para o usuário o significado daquela abreviação, exemplo HTML . para melhorar ainda mais sua semântica é recomendado utilizar a tag <dfn> com um id correspondente e dentro colocar o <abbr> por conta de semanticamente ser mais correto e mais fácil do usuário achar a definição daquela abreviação. O código seria assim: <dfn id="html"> <abbr title="HyperText Markup Language"> HTML </abbr> </dfn>.

Citações

Para utilizar citações no seu texto html existe diversas tags, o <blockquote> aonde seria o elemento pai junto com uma <cite> da fonte aonde você tirou essa citação. por exemplo

O ruim não é ter trabalho. O ruim é ter que trabalhar

Seu Madruga

O código desse exemplo foi <blockquote> <p>O ruim não é ter trabalho. O ruim é ter que trabalhar</p> <cite>Seu Madruga</cite> </blockquote>

Para citações temos também a tag <q>. O elemento <q> indica que o texto dentro da tag é uma pequena citação. Este elemento destina-se a citações curtas que não requerem marcações de parágrafo; para citações maiores use o elemento <blockquote> .

Outras tags

Para utilizar textos pré-formatados se utiliza a tag <pre>, com ela tudo que você modificar no código ela sobe com essa formatação pro browser. Exemplo 

                    Este texto é um texto pré-formatado

Como vimos no exemplo ele puxa até o espaço que se tem nos códigos na hora do browser subir, se eu quebrasse a linha o browser ia ler com quebra de linha também.

temos também a tag <code> que basicamente diz para o browser que aquilo é um código dando uma formatação diferente, ela é bastante usada junto com o JavaScript. Um exemplo em que eu possa usar essas tags é justamente para mostrar um código, como no exemplo abaixo: 

                    
                        .main-nav ul a:hover{
                            background-color: var(--secondary-color-lite) ;
                            padding-right: 1rem;
                            transition: all;
                            transition-duration: .2s;
                            transition-timing-function: linear;
                        }

Basicamente eu abri uma tag <pre> dentro dela abri uma tag <code> e coloquei o código dentro.

Nós também temos a tag <samp>. O elemento <samp> é usado para incluir texto embutido que representa a saída de amostra (ou citada) de um programa de computador, semelhante a tag <code>.

Para indicar ao usuário para ele apertar alguma tecla se utiliza a tag <kbd>, por exemplo CTRL + C. Obviamente com um pouco de CSS fica mais claro para o usuário ainda, mas semanticamente é assim.

Datas e Horários

Para marcar uma data no HTLM se utiliza a tag <time datetime"">, exemplo 03 de Novembro de 2001. Visualmente parece que não mudou nada mas semanticamente esta correto assim. Para adicionar junto com o horário seria <time datetime"ano-mes-diaThora:minuto:segundo">. Lembrando que esse horário é sempre baseado no horário local dos estados unidos para colocar com o fuso-horario do Brasil por exemplo depois do segundo eu coloco -03:00 que seria o fuso-horário aqui do Brasil.
